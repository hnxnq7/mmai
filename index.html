Vision vs. Protocol Effects in EEG2Video:
Baseline Audit and Temporal Alignment on SEED-DV

with Emma Wang and Winston Qian  ·  Team Repo  ·  Project README

EEG-to-video decoding benchmarks like SEED-DV risk confounding genuine visual perception with protocol-induced cognitive artifacts. We audit the EEG2Video framework for run-position artifacts and data leakage, then propose chunk-level latency-aware contrastive alignment to enforce genuine dynamic tracking.

Next Steps — Final Report

Idea 2: Task-Aware Temporal Attention Pooling

Replace mean pooling with per-task attention heads (concept / color / motion) to reveal which EEG timepoints within the 2-second window drive each decoding task. O(T) complexity — avoids the O(T²) self-attention bottleneck we hit with raw EEG.

Idea 3: Chunk-Level Temporal EEG–Video Contrastive Alignment

Partition each 2-second clip into K=4 chunks. Align EEG chunks to frozen VideoMAE video embeddings with a learned biological latency shift δ ∈ {0,1,2} steps (~0–250ms). Evaluated via within-video chunk retrieval (Recall@1, Recall@5).
